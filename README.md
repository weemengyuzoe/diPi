# diPi
Python camera programme code for project d.i.Pi. 

Version used for The Lasalle Show & Final submissions is diPi.py

Previous files were uploaded for documentation purposes

diPi_Advance_WIP.py is a work in progress, advance version of the camera that features a bulb mode function, allowing for long exposure photography.


Programme requires the below to be fully used:
  Raspberry Pi 5
  RPi Camera Module
  Button modules
  Traffic Light LED Module
  Female to Female jumper cables
